Fish game

Horror game where you must collect parts to win and avoid the fishman and the sting rays.

Installation
1) Download the zip file
2) Extract the contents of the zip file into one folder
3) Run horrorgame.exe
Enjoy!

Zip file to download:
https://drive.google.com/drive/folders/1ZaJQY5Zu6l3sm4p2u4lWVmgG35JkjABP?usp=drive_link

Controls:

[WASD] to move

[Shift] to sprint

[space] to jump

[y] to yell
