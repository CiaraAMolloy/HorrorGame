# Bloodletting

Horror game where you must collect parts to win while avoiding the fishman and the sting rays.

Installation
1) Download the zip file
2) Extract the contents of the zip file into one singular folder of your choice
3) Run horrorgame.exe
Enjoy!

## Important Note:
If you win in our game, currently the only way to exit is through using Alt+F4, using Alt+Tab or using the Windows key to exit.

Zip file to download:
https://drive.google.com/drive/folders/1ZaJQY5Zu6l3sm4p2u4lWVmgG35JkjABP?usp=drive_link

Controls:

[WASD] to move

[Shift] to sprint

[space] to jump

[y] to yell
